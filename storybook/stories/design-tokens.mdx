# Design Tokens

This documentation provides an overview of the available theme variables for customizing the appearance of a chat widget. Each variable controls specific elements of the widget's UI, allowing for a consistent and visually appealing user experience.

## Usage

To customize the chat widget's appearance, define these variables within the `:root` selector. Adjust the values as needed to fit your design requirements.

```html
<rasa-chatbot-widget>
  <style>
    :root {
      --color-primary: #ff0080;
      --color-secondary: #ffff80;
      --font-family: 'Brush Script MT', cursive;
    }
  </style>
</rasa-chatbot-widget>
```
